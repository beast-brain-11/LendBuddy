# **LendBuddy – Multilingual Conversational Loan Advisor**  

## **📌 Overview**  
LendBuddy is an AI-powered **multilingual conversational assistant** designed to simplify the **loan application process**, improve **financial literacy**, and ensure **transparency** in loan terms. By leveraging **AI-driven insights**, LendBuddy helps users understand their **loan eligibility**, guides them through the **application process**, and provides **real-time financial advice**—all through a **seamless voice or text-based interface**.  

---

## **🚀 Key Features**  

✅ **AI-Driven Loan Eligibility Check** – Evaluates direct & hidden factors used by top banks.  
✅ **Loan Application Guidance** – Step-by-step assistance for submitting documents and applying for loans.  
✅ **Financial Literacy Tips** – Personalized advice on improving credit scores, saving strategies, and responsible borrowing.  
✅ **Voice & Text Interaction** – Supports **Text-to-Speech (TTS) & Automatic Speech Recognition (ASR)** for easy communication.  
✅ **Transparency & Documentation** – Stores loan details securely for user reference.  
✅ **Loan Provider Ranking & Analysis** – AI-powered ranking system for the best loan options.  
✅ **Terms & Conditions Checker** – Flags hidden clauses and simplifies complex legal terms.  
✅ **Reverse Marketplace** – Users input loan needs, and banks compete with offers.  

---

## **🛠️ Tech Stack**  

### **💻 Backend**  
🔹 **Node.js (Express.js)** – API backend for handling requests.  
🔹 **Redis** – Caching and improving response times.  
🔹 **PostgreSQL** – Database for storing user and loan data.  
🔹 **Firebase** – Authentication and real-time updates.  
🔹 **Python** – AI-related tasks and integrations.  

### **🤖 AI & NLP**  
🔹 **Sarvam AI** – Core AI model for conversational intelligence.  
🔹 **LangChain** – Orchestrating AI-driven responses and tool calling.  

### **🌐 Frontend**  
🔹 **React.js** – Web app interface.  
🔹 **WhatsApp Cloud API** – Chatbot integration.  
🔹 **Telegram, Discord, Slack APIs** – Multi-platform communication.  

### **🔗 APIs & Integrations**  
🔹 **Bank Loan APIs** – Fetching loan eligibility criteria.  
🔹 **Sarvam AI API** – AI-powered conversational support.  
🔹 **Searxng API** – Additional AI-driven features and insights.  

---

## **🔄 How It Works**  

1️⃣ **User Interaction** – Users communicate with LendBuddy via **WhatsApp, Web, or other chat platforms**.  
2️⃣ **Intent Recognition** – AI detects whether the user needs **loan eligibility checks, guidance, or financial advice**.  
3️⃣ **Tool Calling** – The system calls the appropriate service (eligibility check, ranking system, documentation, etc.).  
4️⃣ **Response Generation** – AI provides a **personalized and transparent response** based on real-time data.  
5️⃣ **Loan Application & Reverse Marketplace** – Users can **apply for loans directly** or let banks **compete for their needs**.

## **🔧 Setup & Installation**  

### **1️⃣ Clone the Repository**  
```bash
git clone https://github.com/yourusername/lendbuddy.git
cd lendbuddy
```

### **2️⃣ Install Dependencies**  

#### **Backend Setup (Node.js & Express)**
```bash
cd backend
npm install
```

#### **Frontend Setup (React.js)**
```bash
cd frontend
npm install
```

### **3️⃣ Set Up Environment Variables**  
Create a **.env** file in the backend folder and add:  
```
PORT=5000
REDIS_URL=your_redis_url
DATABASE_URL=your_postgres_url
FIREBASE_CONFIG=your_firebase_config
SARVAM_API_KEY=your_sarvam_ai_api_key
SEARXNG_API_KEY=your_searxng_api_key
```

### **4️⃣ Run the Backend Server**  
```bash
cd backend
npm start
```

### **5️⃣ Run the Frontend**  
```bash
cd frontend
npm start
```
